This is a simple project, it retains around the idea of the Pokedex
To the few unaware the pokedex is a device inside the Pokemon universe in which the Pokedex is a device which identifies the Pokemon you are looking at. 

This IRL pokedex does the same but with animals. 

As stated its very simple, a HTML mock of a simple pokedex, the idetification of the animal is handeled thru Gemini 2.5 flash. 

Pokédex UI: Retro design with a camera viewfinder, status lights, and a digital readout screen.  
Live Camera Feed: With permission opens your device's camera to view wildlife. 
One-Touch Scanning: Captures, compresses, and sends the image to the Gemini API with a single button click.  
Animated Loading Screen: Shows a blinking "Identifying Species..." message while processing.  
Color-Coded Badges: Automatically changes colors based on the animal's type and how endangered it is.  
Full Stat Readouts: Displays the animal's names, habitat, diet, range, lifespan, and size.  
AI Field Notes: Generates a custom bulleted list of fun facts about the captured animal. 
Smart Error Handling: Tells you exactly what went wrong if the camera fails or if no animal was found in the frame.
